# Recovery Companion

A privacy-focused, premium recovery application designed to help track progress for a 6-month challenge.

## Features
*   **6-Month Calendar**: Visual grid to track daily progress.
*   **Strict 9 PM Verification**: Prevents premature check-ins. Daily Armor during the day, Check-in at night.
*   **Dynamic Rewards**: Motivational Videos, Quotes, and Bible Verses.
*   **Privacy First**: All data is stored locally on your device.

## How to Run Locally

### Prerequisites
You need **Node.js** installed on your computer.

### Steps
1.  Open your terminal (Command Prompt, PowerShell, or Terminal).
2.  Navigate to this folder:
    ```bash
    cd d:\Antigravity\Website
    ```
3.  Install dependencies (only need to do this once):
    ```bash
    npm install
    ```
4.  Start the application:
    ```bash
    npm run dev
    ```
5.  Open your browser and visit the URL shown (usually `http://localhost:5173`).
